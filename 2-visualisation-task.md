# Group Work Visualisation

In this group work you will work with different plotting libraries. Some of the most commonly used or interesting libraries in recent years are [Matplotlib](https://matplotlib.org/index.html), [Seaborn](https://seaborn.pydata.org/), [Plotly](https://plotly.com/python/) and [Altair](https://altair-viz.github.io/). Today you will work with all of them to get a first impression of their advantages, disadvantages and differences. 


## Task & Objectives

Before working with the plotting libraries create a new virtual environment using the [requirements.txt](requirements.txt) file. It contains all the plotting libraries you will need for this task.

The task for each group is to create **four appealing plots** using **each of the four plotting libraries**. Each group will create a different type of plot using different datasets (see table below). At the end of the group work, you should have a well-structured Jupyter notebook that contains the Python code to import the libraries and create the plots, as well as some explanations. You will share the notebooks with the other groups so they can serve as cheat sheets for your further Data Science journey. We've already added a [template notebook](2-visualisation-notebook-template.ipynb) to the repo to make it easier for you to get started. 

Whether a plot is appealing or not is also always a matter of taste, but the following key points should always be considered:

* Title, labels, and text in general should be readable. 
* Don't forget the units!
* When using different colors, be aware that some people suffer from (various types of) colorblindness. 
* A plot should be easy to understand. Don't overwhelm your stakeholder/audience with too much details/information/artistic creativity.
